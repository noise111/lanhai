统一小区设置
小区设置 说明 是否启用
人工审核 开启住户审核后，用户绑定需要管理人员审核方可使用物业相关功能，默认不开启审核
手机号验证 开启手机号验证后，移动端粉丝注册的时候，通过输入手机号码后，进行验证手机号是否已导入系统，建议配合短信验证码来验证当前录入的手机号业主是否为真正的业主
楼宇信息验证 开启楼宇信息验证，会和预先后台生成的楼宇，区域，房屋进行验证是否存在
楼宇联动选择 开启楼宇联动选择，会根据预先后台录入的楼宇，单元，房号信息，业主进行选择绑定
注册码验证 开启发放的注册码来给业主完成注册，减少业主需要输入的资料。如同时开启手机号验证，将同时验证手机号和注册码是否在系统中同时存在
游客浏览 开启游客浏览权限后，粉丝可直接进入小区主页，无需注册，需到菜单设置中设置每个菜单权限。
房屋托管 开启房屋托管后，移动端粉丝发布的手机号码不在显示，会显示托管电话
报修审核 开启报修审核后，粉丝发布的报修，需要后台审核后，才可显示
建议审核 开启建议审核后，粉丝发布的建议，需要后台审核后，才可显示
集市审核 开启集市审核后，粉丝发布的集市，需要后台审核后，才可显示
租赁审核 开启租赁审核后，粉丝发布的房屋租赁，需要后台审核后，才可显示
拼车审核 开启拼车审核后，粉丝发布的拼车信息，需要后台审核后，才可显示
报修手机端抢单 开启抢单模式，手机端报修接收人将启用抢单，第一个点击，就是抢单成功。
建议手机端抢单 开启抢单模式，手机端建议接收人将启用抢单，第一个点击，就是抢单成功。
报修全部显示 开启报修全部显示后，粉丝发布的报修信息在微信端就在当前小区全部显示
建议全部显示 开启建议全部显示后，粉丝发布的建议信息在微信端就在当前小区全部显示
微信开门 开启的话，业主注册默认就有微信开门权限
授权验证开门 默认微信开门采用的业主注册的楼栋单元匹配，进行开门。开启授权验证开门后，需要先在后台设置业主授权开门
微信开门关联物业费 此设置与微信开门硬件有关，开启此设置后，会关联业主的物业费
新增小区默认勾选 新增小区时公告、幻灯、魔方、广告会默认勾选该小区，开启时新增小区不会在该小区显示
