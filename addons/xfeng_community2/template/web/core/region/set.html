   小区设置
小区设置 说明 是否启用
报修审核 开启报修审核后，粉丝发布的报修，需要后台审核后，才可显示
建议审核 开启建议审核后，粉丝发布的建议，需要后台审核后，才可显示
二手审核 开启二手审核后，粉丝发布的二手信息，需要后台审核后，才可显示
租赁审核 开启租赁审核后，粉丝发布的租赁信息，需要后台审核后，才可显示
拼车审核 开启拼车审核后，粉丝发布的拼车信息，需要后台审核后，才可显示
报修手机端抢单 开启抢单模式，手机端报修接收人将启用抢单，第一个点击，就是抢单成功。
建议手机端抢单 开启抢单模式，手机端建议接收人将启用抢单，第一个点击，就是抢单成功。
报修全部显示 开启报修全部显示后，粉丝发布的报修信息在微信端就在当前小区全部显示
建议全部显示 开启建议全部显示后，粉丝发布的建议信息在微信端就在当前小区全部显示
房屋托管 开启房屋托管后，移动端粉丝发布的手机号码不在显示，会显示托管电话
微信开门 开启的话，业主注册默认就有微信开门权限
授权验证开门 默认微信开门采用的业主注册的楼栋单元匹配，进行开门。开启授权验证开门后，需要先在后台设置业主授权开门
微信开门关联物业费 此设置与微信开门硬件有关，开启此设置后，会关联业主的物业费
是否开启小区收入显示 开启后，将在微信端个人中心，展示小区的总收入
是否开启小区注册 默认可以注册，开启后用户不可注册
是否开启小区报修处理流程 开启后用户可查看报修处理流程
是否开启小区建议处理流程 开启后用户可查看建议处理流程
