统一小区设置
小区设置 说明 是否启用
报修审核 开启报修审核后，粉丝发布的报修，需要后台审核后，才可显示
建议审核 开启建议审核后，粉丝发布的建议，需要后台审核后，才可显示
报修手机端抢单 开启抢单模式，手机端报修接收人将启用抢单，第一个点击，就是抢单成功。
建议手机端抢单 开启抢单模式，手机端建议接收人将启用抢单，第一个点击，就是抢单成功。
报修全部显示 开启报修全部显示后，粉丝发布的报修信息在微信端就在当前小区全部显示
建议全部显示 开启建议全部显示后，粉丝发布的建议信息在微信端就在当前小区全部显示
报修回复模板 例如：我们已收到您的报修，请等待我们的维修员联系您！
建议回复模板 例如：我们已收到您的建议，请等待我们的管理员联系您！
新增小区默认勾选 新增小区时公告、幻灯、魔方、广告、家政项目、底部菜单会默认勾选该小区，开启时新增小区不会在该小区显示
